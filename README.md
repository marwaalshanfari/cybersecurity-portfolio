üîπ Project 1: GAN-Based Data Balancing for Network Intrusion Detection

- Objective:
To address the class imbalance problem in network intrusion detection systems by generating high-quality synthetic attack traffic and evaluating its impact on detection performance.

- Tools Used:
Python, TensorFlow, SDV (CTGAN, TVAE), Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn

- Methodology:

Created an imbalanced network traffic dataset representing normal and attack flows.

Implemented and trained three generative models: Vanilla GAN, CTGAN, and TVAE to synthesize attack traffic.

Balanced the dataset using synthetic samples generated by each model.

Trained a Random Forest classifier and evaluated performance using Accuracy, Precision, Recall, F1-Score, and AUC.

Compared models to demonstrate the accuracy paradox in imbalanced security data.

- Outcomes:

CTGAN produced the most realistic synthetic data for mixed tabular network features.

Improved attack detection performance, particularly in Recall and F1-Score, which are critical for security systems.

Gained strong hands-on experience in applying GANs for cybersecurity analytics and evaluating ML models on imbalanced datasets.
